## 3. 수동으로 해야 할 작업

### 3.1 필수 (반드시 해야 함)

1. **`.env` 파일 생성**
   ```bash
   cp .env.example .env
   ```
   - `GEMINI_API_KEY`: Gemini API 키 입력
   - `LAW_API_KEY`: 국가법령정보센터 API 키 입력

2. **의존성 설치**
   ```bash
   pip install -r requirements.txt
   ```

3. **앙커 데이터 입력** (`data/anchors.json`)
   - 실제 시음 기반으로 10~20개 앙커 데이터 입력

### 3.2 권장 (해야 함)

4. **데이터 파이프라인 실행**
   ```bash
   python app/auto_pipeline.py
   ```

5. **서버 실행**
   ```bash
   uvicorn app.main:app --reload --port 8000
   ```

6. **API 테스트**
   - http://localhost:8000/docs

## 4. API 엔드포인트

| 엔드포인트 | 메서드 | 설명 |
|------------|--------|------|
| `/` | GET | 루트 |

| `/api/law/filter` | POST | 콘텐츠 법률 필터링 |
