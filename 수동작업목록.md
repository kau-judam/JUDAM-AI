# 주담 AI 서버 - 수동 작업 목록

## 1. 환경 설정

### 1.1 .env 파일 생성
- ✅ `.env.example` 생성 완료
- ⚠️ `.env` 파일 생성 필요 (실제 API 키 입력)
  - `GEMINI_API_KEY`: Gemini API 키 발급 필요
  - `LAW_API_KEY`: 국가법령정보센터 API 키 발급 필요
  - `DATABASE_URL`: PostgreSQL DB 연결 정보
  - `REDIS_URL`: Redis 연결 정보

### 1.2 의존성 설치
- ✅ `requirements.txt` 생성 완료
- ⚠️ `pip install -r requirements.txt` 실행 필요

## 2. 데이터 준비

### 2.1 앙커 데이터 입력 (data/anchors.json)
- ❌ 실제 시음 기반으로 10~20개 앙커 데이터 입력 필요
- 각 앙커는 다음 필드 포함:
  - name: 전통주 이름
  - sweetness, body, carbonation, flavor, alcohol, acidity, aroma_intensity, finish: 0~10 점
  - note: 시음 기반 설명

### 2.2 원본 CSV 데이터 준비 (data/raw/)
- ✅ CSV 파일 2개 존재:
  - 한국농수산식품유통공사_전통주정보_20241231.csv
  - RT_KOREAN_LIQUOR_INFO_202407.csv

### 2.3 데이터 파이프라인 실행
- ✅ `app/auto_pipeline.py` 생성 완료
- ⚠️ `python app/auto_pipeline.py` 실행 필요
